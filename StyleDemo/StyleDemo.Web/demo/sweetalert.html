弹框说明
属性 默认值 描述
title null 弹窗标题
text null 弹窗内容
type null 弹窗类型，默认Null，加上该值后会有对应的logo显示"warning", "error", "success" ,"info"
allowOutsideClick false 允许用户点击非弹窗部分后关闭弹窗
showCancelButton false 如何设置为true，一个取消按钮将会显示，点击它会关闭弹窗
showConfirmButton true 如何设置为false，这个确认按钮会被隐藏。如果你设置了一个false，确保你设置了一个定时器或者让allowOutsideClick为true来用于关闭弹窗
confirmButtonText "OK" 弹窗确认按钮上的文字
confirmButtonColor #1ab394 确认按钮的颜色
cancelButtonText "Cancel" 取消按钮的文字
timer null 自动关闭弹窗的时间(毫秒)1000为1S
animation true 如何设置为false，弹窗就没有动画，默认值为pop，还有slide-from-top, slide-from-bottom方式
inputType text 改变input标签输入的类型当type属性为input的时候
inputPlaceholder null placeholder
inputValue null input框的默认值（type为input的情况下）
closeOnConfirm true 如何设置为false,才可以点击按钮后再执行一个function，适用于确认按钮
closeOnCancel true 和确认的一样，这个只能适用于取消按钮
callback null 点击确认按钮后所执行的函数 
                                swal({
                                    title: "您确定要删除这条信息吗",
                                    text: "删除后将无法恢复，请谨慎操作！",
                                    type: "warning",
                                    showCancelButton: true,
                                    confirmButtonColor: "#DD6B55",
                                    confirmButtonText: "删除",
                                    closeOnConfirm: false
                                }, function () {
                                    swal("删除成功！");
                                });
基本alert 带标题和内容 success内容 警告并带确认按钮操作 
swal("你好");
 
swal("Here's a message!", "It's pretty, isn't it?")
或者
swal({
    title:"Here's a message!",
    text:"isn't it?"
})
                            
swal("Here's a message!", "It's pretty","success")
或者
swal({
    title:"Here's a message!",
    text:"isn't it?",
    type:"success"
})  
                            
swal({ 
    title: "Are you sure?", 
    text: "You will not be!",
    type: "warning",
    showCancelButton: true,
    confirmButtonColor: "#DD6B55",
    confirmButtonText: "Yes, delete it!",
    closeOnConfirm: false },
    function(){

    });
