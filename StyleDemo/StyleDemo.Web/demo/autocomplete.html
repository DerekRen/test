DEMO代码
//初始化
var testObj = $("#autocomplete").tcAutoComplate({
    'placeHolder':'请输入右侧数据',
    'ajaxUrl': 'src/data/autocomplete.json',
    'ajaxKey':'searchName',
    'returnTxtKey':'Name',
    'returnIDKey':'ID',
    'fn':function(data){
    	$("#result").html(JSON.stringify(data));
    },
    'clearFn':function(){
    	$("#result").html("");
    }
});

API
属性说明
属性名称 说明 默认值
placeHolder placeHolder ""
ajaxUrl 获取列表的异步地址（必填） ""
ajaxKey 获取列表的搜索关键字参数（必填） ""
minCharacters 出发自动补齐的最小字符数 1
parameters 除了关键字参数之外的参数（json对象） null
returnTxtKey 异步返回列表元素的文本的属性名（示例中的Name） dataName
returnIDKey 异步返回列表元素的ID的属性名（示例中的ID） dataID
defaultTxt 默认显示的文本 ""
defaultObj 默认值对应的对象
（{"Name": "测试客户111a","ID": "11067"}）		 null
initFn 搜索前需要执行的方法 null
fn 选择列表项之后的回调 null
clearFn 清除输入框之后的回调 null
方法说明（均无返回值）
方法名称 说明 参数
setDefaultValue 设置默认值 txt(string),dataObj(object)
对应属性中的defaultTxt，defaultObj
setOtherParameters 设置异步的其他参数 parametersObj(object)
对应属性中的parameters
clearInput 清空搜索框 -
