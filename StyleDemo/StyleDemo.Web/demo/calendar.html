API
属性名称 说明 默认值
monthNum 日历个数 2
style 日历的称呈现方式,默认是"pick"（日期选择）,可配置为"show"，用于呈现数据，需要同时配置wrapper属性，作为日历的父元素 ’pick‘
wrapper 日历的父元素选择器，通常和style设置为"show"配合使用 ""
firstDay 日历面板上面从星期几开始，默认是0，从星期日开始 0
如果设置了style是pick的话，就需要写对应的方法，pick下的属性
startDate 设置日历的可选区域的起始时间，默认是1900-1-1 1900-1-1
endDate 设置日历的可选区域的终止时间，默认是2100-12-31 2100-12-31
elem 与日历关联的元素选择器，在"pick"方式，用作定位参照元素，如果elem是input的元素，日历单击时会把日期值传递给该input框 ""
mode 选择的模式：默认"rangeFrom"，表示选择起始时间或者单个时间, 可配置为"rangeTo"，表示选择结束时间，在currentDate中配置了起始时间，会有区间选择效果 "rangeFrom"
currentDate currentDate设置为数组时，第1个值表示起始时间，第2个表示终止时间，设置为日期字符串或日期对象时，表示起始时间 null
offset 自定义偏移，支持top和left属性，除了特殊情况使用，通常采用自动定位就好 null
代码示例
var cal = new $.Calendar({
monthNum:"1" 	//单历写1，双历写2
});
var endDate = new Date();
endDate.setMonth(endDate.getMonth() + 12);//创建一个结束的日期并大于开始日期12个月，就是最大选到12个月以内
$("#startTime1").on("focus", function(e) {
    cal1.pick({
        elem: this,  // 如果设置了elem的值，且elem参数为input框
        startDate: new Date(),//开始日期New Date
        mode: "rangeFrom",
        endDate: endDate,
        currentDate: [new Date()]
    });
});

下面代码是选结束日期使用的，如果不需要的话是不需要写的

$("#endTime1").on("focus", function(e) {
    var startDate = new Date($("#startTime1").val());//让开始日期等于上面选的日期
    endDates = new Date(startDate);
    endDates.setMonth(endDates.getMonth() + 12);
    cal1.pick({
        elem: this,  // 如果设置了elem的值，且elem参数为input框
        startDate: startDate,
        mode: "rangeFrom",
        endDate: endDates,
        currentDate: [startDate]
    });
});
